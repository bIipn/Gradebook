# Gradebook Manager

**A simple yet extensible C# console application for managing student grades, calculating averages, and generating reports.**

Built to demonstrate clean object-oriented design, data handling, and basic statistics in .NET â€” perfect for learning or as a foundation for more advanced grade-tracking tools.

## Features
- Add/remove students and assignments
- Record and update grades
- Calculate weighted averages, letter grades, and GPA
- Generate summary reports (class average, top performers)
- Input validation and error handling

## Tech Stack
- **C#** (.NET 8+ recommended)
- Console Application
- Object-Oriented Programming (classes for Student, Assignment, Gradebook)
